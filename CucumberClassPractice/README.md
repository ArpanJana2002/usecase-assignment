# CucumberClassPractice

## Overview
CucumberClassPractice is a Java-based project that uses Maven as the build tool. It integrates Selenium, Cucumber, TestNG, and other libraries to facilitate behavior-driven development (BDD) and automated testing.

## Features
- **Cucumber**: For BDD-style test scenarios.
- **Selenium**: For browser automation.
- **TestNG**: For test execution and reporting.
- **WebDriverManager**: For managing browser drivers automatically.
- **ExtentReports**: For generating detailed test reports.

## Prerequisites
- **Java 17** or higher
- **Maven** (Ensure it is installed and added to your system's PATH)
- **IntelliJ IDEA** (Recommended IDE)

## Project Structure
- `src/main/java`: Contains the main application code.
- `src/test/java`: Contains the test code, including Cucumber step definitions and TestNG test runners.
- `pom.xml`: Maven configuration file for managing dependencies and plugins.

## Dependencies
The project uses the following dependencies:
- Selenium: `4.21.0`
- Cucumber: `7.18.1`
- TestNG: `7.10.2`
- WebDriverManager: `5.8.0`
- SLF4J: `2.0.13`
- ExtentReports Cucumber Adapter: `1.14.0`
